# 🔎Objetivo
Este projeto foi criado com o objetibo de avaliar as opiniãos de 50 clientes, sobre o atendimento de uma empresa fictícia TudoWeb. Cada cliente entrevistado terá de colocar seu nome, idade e escolher seu grau de satisfação entre: Excelente(1), Bom(2) e Ruim(3).

Ao final da pesquisa, será analizada a quantidade de respostas "Excelentes" e "Ruins".s

## ⚙️Estrutura
 - For: para poder repetir o questionário 50 vezes
 - +=: para acumular os resultados gerados
 - If e Ilif: para verificar as condições, de acordo com as respostas dos clientes
